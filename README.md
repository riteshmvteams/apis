1. [Api Documentation](https://documenter.getpostman.com/view/30139783/2sA3rxosJF)
2. [Api url](https://apis-f4f8.onrender.com)

## Run Locally

1. Clone the Repo

```bash
git clone https://github.com/riteshmvteams/apis.git
```

2. Go into project

```bash
cd apis
```

3. Install the Packages

```bash
npm install
```

4. Run Local

```bash
npm run dev
```
